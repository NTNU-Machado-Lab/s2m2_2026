# S2M2 2026

Exercises for the [S2M2 summer school 2026](https://s2m2.bio.di.uminho.pt/).

### Instructions

You can run this tutorial online using binder: 

👉 [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/NTNU-Machado-Lab/s2m2_2026/HEAD?urlpath=%2Fdoc%2Ftree%2Findex.ipynb)

Or you can run locally on your own computer:

```
git clone https://github.com/NTNU-Machado-Lab/s2m2_2026
cd s2m2_2026/
conda env create --file environment.yml
conda activate s2m2
jupyter lab
```
